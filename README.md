# 🌿 Hypnosystemischer Coach

Eine interaktive Coaching-App basierend auf dem hypnosystemischen Ansatz nach Gunther Schmidt.

## Features

- **Hypnosystemisches Coaching** mit strukturierter Auftragsklärung
- **Fallverwaltung** - mehrere Themen mit Sitzungen verwalten
- **Bikablo-Skizzen** - visuelle Mind-Maps im Flipchart-Stil
- **Export** - Word, HTML oder Text
- **Backup/Import** - Daten sichern und wiederherstellen
- **Responsive** - funktioniert auf Desktop und Mobil

## Setup

Siehe [SETUP-ANLEITUNG.md](SETUP-ANLEITUNG.md) für die vollständige Einrichtung.

**Kurzfassung:**
1. Cloudflare Worker einrichten (API-Proxy)
2. Worker-URL in `index.html` eintragen
3. Auf GitHub Pages deployen

## Technologie

- React 18 (via CDN)
- Anthropic Claude API (Sonnet)
- Cloudflare Workers (API-Proxy)
- GitHub Pages (Hosting)
- localStorage (Datenspeicherung)
